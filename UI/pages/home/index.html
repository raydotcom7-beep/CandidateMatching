यह कैसे काम करता है

3 आसान चरण

  1. 1

    आधार OTP सत्यापन

    अपना आधार नंबर दर्ज करें। UIDAI से OTP प्राप्त करें और eKYC से आपकी जानकारी स्वतः भर जाएगी।

  2. 2

    जानकारी सत्यापित करें

    व्यक्तिगत विवरण, व्यवसाय और दस्तावेज़ अपलोड करें। सभी डेटा AES-256 से सुरक्षित है।

  3. 3

    डिजिटल कार्ड प्राप्त करें

    स्वीकृति के बाद अपना डिजिटल रोजगार कार्ड और अद्वितीय Rojgar ID तुरंत डाउनलोड करें।
